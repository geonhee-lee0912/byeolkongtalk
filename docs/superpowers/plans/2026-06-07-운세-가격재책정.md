### Task 1: 오늘의 타로 무료 횟수 뱃지 (Part 1)

**Files:**
- Modify: `app/fortune/page.tsx`

참고 — 현재 관련 코드:
- 상태/페치 (line 24–34):
  ```tsx
  export default function FortunePage() {
    const [tab, setTab] = useState<FortuneTab>("saju");
    const [daily, setDaily] = useState<DailyStatus | null>(null);
    const items = FORTUNE_LIST.filter((f) => f.base === tab);

    useEffect(() => {
      void fetch("/api/fortune/daily-status", { cache: "no-store" })
        .then((r) => (r.ok ? r.json() : null))
        .then((d) => d && setDaily(d))
        .catch(() => {});
    }, []);
  ```
- 뱃지 (line 91–105, `items.map((f) => {` 안, `const inner = (` 내부):
  ```tsx
  {f.cost === 0 ? (

        무료 소진 · ⭐ {daily.nextCost}

  ```

`DailyStatus` 인터페이스(`{ used; limit; remaining; nextCost }`)는 두 status API가 동일 형태를 반환하므로 그대로 재사용한다. `/api/fortune/tarot-daily-status`는 이미 존재하며 `{ used, limit, remaining, nextCost }`를 반환한다(신규 API 불필요).

- [ ] **Step 1: `tarotDaily` 상태 추가**

`app/fortune/page.tsx`의 `const [daily, setDaily] = useState<DailyStatus | null>(null);` 바로 아래 줄을 추가한다:

```tsx
  const [tarotDaily, setTarotDaily] = useState<DailyStatus | null>(null);
```

- [ ] **Step 2: `tarot-daily-status` 병렬 fetch 추가**

`useEffect` 안, 기존 daily fetch `.catch(() => {});` 바로 다음에 추가한다(같은 effect 내):

```tsx

      .then((r) => (r.ok ? r.json() : null))
      .then((d) => d && setTarotDaily(d))
      .catch(() => {});
```

- [ ] **Step 3: map 콜백에서 `freeStatus` 계산**

`{items.map((f) => {` 직후, `const inner = (` 라인 바로 앞에 추가한다:
