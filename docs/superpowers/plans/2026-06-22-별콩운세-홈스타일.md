# 별콩 운세 홈 스타일 카드·내용 구성 통일 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 별콩 운세(`/fortune`) 목록 카드를 홈(고민 상담) 카드 구성(그라데이션 타일 + 설명 + 해시태그)으로 바꾸고, "이렇게 사용해요" 안내 박스를 추가한다.

**Architecture:** `lib/fortune/types.ts`에 운세별 그라데이션·해시태그 맵을 추가하고, `app/fortune/page.tsx`에서 안내 박스 + 카드 렌더를 홈 스타일로 교체한다. 사주/타로 탭·비용 배지·흰 카드 표면은 유지.

**Tech Stack:** Next.js 16, TypeScript, Tailwind v4.

> **검증:** 단위 테스트 없음. 태스크별 `npx tsc --noEmit` + 마지막 `npm run build` + dev 브라우저 확인.

## Task 3: 운세 카드 홈 스타일로 (그라데이션 타일 · 설명 · 해시태그 · chevron 제거)

**Files:**
- Modify: `app/fortune/page.tsx`

- [ ] **Step 1: 타일 배경을 그라데이션으로**

현재:
```tsx
              <div className="w-12 h-12 rounded-xl bg-lilac-soft/60 flex items-center justify-center text-[24px] shrink-0">
                {f.type === "saju_full" ? <RedHorseIcon size={30} /> : f.emoji}
              </div>
```
교체 (단색 `bg-lilac-soft/60` 제거, 인라인 그라데이션):
```tsx
              <div
                className="w-12 h-12 rounded-xl flex items-center justify-center text-[24px] shrink-0"
                style={{ background: FORTUNE_GRADIENTS[f.type] }}
              >
                {f.type === "saju_full" ? <RedHorseIcon size={30} /> : f.emoji}
              </div>
```

- [ ] **Step 2: 설명 강조 + 해시태그 칩 추가**

현재 (tagline `<p>` 만 있음):
```tsx
                <p className="text-[12px] text-text-light/80 mt-1 leading-snug line-clamp-2">
                  {f.tagline}
                </p>
              </div>
```
교체 (`text-[12.5px]` 로 강조 + 해시태그 칩 블록 추가):
```tsx
                <p className="text-[12.5px] text-text-light/80 mt-1 leading-snug line-clamp-2">
                  {f.tagline}
                </p>
                <div className="flex flex-wrap gap-1 mt-2">
                  {FORTUNE_HASHTAGS[f.type].map((h) => (
                    <span
                      key={h}
                      className="text-[11px] font-bold text-lilac-deep bg-lilac-soft/60 px-2 py-0.5 rounded-full"
                    >
                      #{h}
                    </span>
                  ))}
                </div>
              </div>
```

- [ ] **Step 3: 활성 카드 chevron 제거 (비활성 "준비 중"은 유지)**

현재:
```tsx
              {f.active ? (
                <span className="text-text-light/40 text-lg shrink-0">›</span>
              ) : (
                <span className="text-[10px] text-text-light/50 shrink-0">준비 중</span>
              )}
```
교체 (활성은 아무것도, 비활성만 "준비 중"):
```tsx
              {!f.active && (
                <span className="text-[10px] text-text-light/50 shrink-0">준비 중</span>
              )}
```

- [ ] **Step 4: 타입 컴파일 검증**
