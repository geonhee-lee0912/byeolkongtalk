# 별콩톡 전체 가격 리밸런싱 설계

작성일: 2026-06-03
상태: 설계 합의 완료 (구현 전)

## 배경 / 목표

현재 제품 가격과 충전 패키지가 누적적으로 붙어와 정합성이 깨져 있다. 이번 작업은 **체감 가치 중심(value-based)** 으로 전체를 재설계한다. 원가(API 비용)는 제약이 아니다 — 모든 제품이 80%+ 마진이라 가격은 "이 결과물에 얼마를 낼 만한가"로 결정한다.

환율 기준: **1,500원/$**. 모델: `claude-sonnet-4-20250514`, 출력 단가 $15/M, 입력 정적 블록 캐싱(read 0.1×).

핵심 원칙:
1. **별 앵커 1별 = 100원 유지** — 모든 가격 직관의 기준.
2. **빈도가 가격 민감도를 결정** — 반복 소비 제품은 싸게, 연 1회 특별 소비는 초프리미엄.
3. **고민 상담과 리포트(운세)는 별개 제품군** — 상담은 대화형(가볍게 묻기), 리포트는 one-shot 심층. 가격 역전(상담 20별 < 사주분석 50별)은 역전이 아니라 다른 제품군.

---

## Section 1 — 충전 패키지 재설계

| 패키지 | 현재 | 제안 | 정가 대비 할인 |
|---|---|---|---|
| 10별 | 1,000원 | **1,000원** | 0% |
| 35별 → 30별 | 2,900원 | **2,800원** | 7% |
| 80별 → 70별 | 5,900원 | **5,900원** (히어로) | 16% |
| 150별 | 9,900원 | **11,000원** | 27% |
| 230별 → 300별 | 12,900원 | **19,900원** | 34% |

- 별 개수를 깔끔한 단위(30/70/150/300)로 정렬 → 제품 가격(5/10/15/20/25/40/50별)과의 조합 직관 개선.
- 할인 곡선 0 / 7 / 16 / 27 / 34% — 단조 증가, 히어로(70별)에 시각적 강조.
- **최대 할인 44% → 34% 축소**: 헤비 유저의 실효 단가 약 18% 인상. 라이트 유저(소액 패키지)는 거의 동일.

## Section 2 — 제품 가격 (가치 밴드)

| 밴드 | 제품 | 현재 | 제안 |
|---|---|---|---|
| **가볍게** | 오늘의 운세 | 무료×5 → 7별 | 무료×5 → **5별** |
| | 타로 원카드 | 10별 | **10별** |
| **깊게** | 이번달 운세 | 15별 | **15별** |
| | 타로 투카드 | 15별 | **15별** |
| | 사주 상담 | 20별 (flat) | **20별** (flat 유지) |
| **프리미엄** | 타로 쓰리카드 | 22별 | **25별** |
| | 타로 관계(5장) | 35별 | **40별** |
| **초프리미엄** | 2026 사주분석 | 30별 | **50별** |

이동 근거:
- **오늘의 운세 소진 후 7→5별**: 진입 미끼 마찰 최소화. 30별 패키지 = 6회로 단위 직관 깔끔.
- **프리미엄 인상(쓰리카드 +3)**: 투카드(15)와 간격을 벌려 "한 단계 더 깊게"를 가격으로 표현.
- **관계 스프레드 35→40별**: 5장 풀 해석, 최고가 일반 제품.
- **사주 상담 20별 flat 유지**: 상담은 리포트와 별개 제품군. 깊이별 차등(스프레드화)은 이번 범위 밖 — 별도 기획.

## Section 3 — 사주분석 초프리미엄화 + 리포트 분량 강화

### 전략

2026 사주분석은 **연 1회 볼까 말까 한 특별 소비**다. 비교 앵커가 다른 AI 리포트가 아니라 오프라인 철학관/만세력(3만~10만원). 따라서 가격 민감도가 거의 없어 초프리미엄 전략이 유효하다. 단:
- **브랜드 톤 제약**: 별콩이는 "전문가"가 아니라 "친구". 오프라인 가격(수만원)까지 가면 기대 갭/역풍. → **충동구매 범위(커피 한두 잔, 5,000원)** 안에 둔다.
- **분량이 값을 해야 함**: 가격만 올리고 리포트가 얇으면 신뢰 붕괴.

### max_tokens 차등

현재 모든 one-shot 리포트(`generateOnce()`)가 `max_tokens: 2048` 단일값. 제품별로 차등한다.

| 제품 | 현재 | 제안 | 체감 분량 |
|---|---|---|---|
| 오늘의 운세 | 2048 | **1536** | 짧고 명료 (단답) |
| 이번달 운세 | 2048 | **2560** | 섹션별 적당히 |
| 2026 사주분석 | 2048 | **8192** | 풀 리포트 6~8섹션 (약 4,000~5,000자) |

오늘의 운세는 오히려 줄여 — 저가/반복 제품이라 "한 장 정리" 컨셉에 간결함이 맞다.

### 사주분석 리포트 구조

6~8개 섹션으로 확장: **총운 · 재물 · 애정/관계 · 직업/진로 · 건강 · 월별 흐름** 등. 단순 `max_tokens` 상한만 올리면 모델이 짧게 끝낼 수 있으므로, `buildFortuneSystem`의 dynamic 섹션 가이드(saju_full 분기)에 **섹션 목록 + 각 섹션 최소 분량 지시**를 추가한다.

### 마진 검증 (환율 1,500원/$)

- **2026 사주분석**: 출력 2048→8192 토큰 = +6144 × $15/M × 1,500 ≈ +138원. 기존 ~54원 → 약 **210원**. 50별 = 5,000원 → 마진 **~96%** ✅
- **이번달 운세**: 출력 2048→2560 = +12원 → 약 48원. 15별 = 1,500원 → 마진 **~97%** ✅
- **오늘의 운세**: 분량 축소로 원가 더 하락.

one-shot 리포트는 입력(페르소나)이 캐시되고 출력 토큰 단가가 작아, **분량을 4배로 늘려도 마진은 96%+ 유지**된다.
