# 별콩톡 유료 광고 시작 전략 설계

작성일: 2026-07-04
상태: 설계 합의 완료 (구현 전)

> 첫 유료 광고를 "싸게, 우리 실제 숫자부터 확인"하는 소액 학습 전략. 매체·예산·오퍼·측정·소재까지 실행 직전 수준으로 확정.

## 배경 / 목표

별콩톡(사주+타로 AI 상담)의 첫 유료 광고. 사용자는 광고 초보 → 복잡한 멀티채널 대신 **한 매체 집중 + 소액 학습**으로 시작한다.

- **1순위 목표**: 결제(매출/전환).
- **현실 제약**: 일 예산이 작아 결제(월 5~16건 수준)로는 메타 알고리즘이 학습을 못 함. 그래서 **광고 최적화 이벤트는 "가입/체험", 결제는 지표로 관찰**한다.
- **핵심 인식 (원가 계산 근거)**: 국내 메타 기준 추정(클릭당 ~₩700, 클릭→가입 25%, 가입→결제 5%)으로 결제 1건당 CAC ≈ ₩56,000. 첫 결제 객단가(~₩4,000)보다 훨씬 커서 **순수 성과광고로 첫 결제만 노리면 적자**. 회수는 **재구매(LTV) + 공유(바이럴)** 로 한다. 이 소액 테스트의 목적은 **우리 실제 CPC·가입률·결제율·CAC를 측정**하는 것.

## 핵심 결정 요약

| 항목 | 결정 |
|---|---|
| 목표 | 결제. 단 광고 최적화는 가입/체험, 결제는 관찰 |
| 예산 | **일 ₩10,000** (월 약 30만원), 소액 학습 |
| 매체 | **인스타그램/메타 단일 집중** (틱톡·카카오·네이버·유튜브는 나중) |
| 전략 골격 | **A. 순수 성과광고** (바이럴 B는 CAC 확인 후 얹음) |
| 오퍼 | 가입 시 **웰컴 별**(간판 리딩 1회 무료) + **첫 결제 보너스 별** |
| 소재 | 3종 — 고민상담 / 무료 오늘의 운세 / 직접 뽑는 타로 |
| 타깃 | 한국·20~39, **메타 Advantage+ (broad)** 로 시작 |
| 캠페인 구조 | 1 캠페인 · 1 애드셋 · 소재 3종 (분산 금지) |
| 측정 | 메타 Pixel + Conversions API, 이벤트 가입/체험완료/결제 |

## 1. 퍼널 & 랜딩

광고 클릭 → **소재별 딥링크(의도 일치)** → 로그인 → 무료 리딩 → 결과 → (첫) 결제.

- 광고 유입은 기존 홈(`/` 감정 태그 그리드)보다 **전용 "무료 훅" 진입**으로 마찰 최소화. 소재별로 딥링크를 나눈다:
  - 고민상담 소재 → 로그인 → 고민 입력 → 웰컴 별로 무료 상담
  - 무료 오늘의 운세 소재 → 오늘의 운세(이미 무료) → 이후 이번달·사주분석·상담 업셀
  - 직접 뽑는 타로 소재 → 타로 draw 흐름 → 웰컴 별로 무료
- 모든 유입 링크에 **UTM 파라미터**(source/medium/campaign/content=소재명) 부착.

## 2. 오퍼 & 필요한 제품 변경 (개발 항목)

광고 훅을 성립시키기 위해 제품에 다음을 추가한다.

- **신규 웰컴 별 지급**: 카카오 최초 가입 시 **30별** 지급. 사주 상담(20)·타로 원/투/쓰리카드(10/15/25)·이번달 운세(20) 등 대부분의 단품 리딩 1회를 커버(관계 스프레드 40·2026 사주분석 60은 제외). 오늘의 운세는 이미 무료라 웰컴 별은 상담·타로에서 주로 쓰임. `star_transactions`에 `source=welcome_bonus`, **멱등(유저당 1회)**. 무료 리딩 1회의 실제 원가는 Claude API ~₩50~250로, ₩56,000 CAC 대비 사실상 공짜 총알.
- **첫 결제 혜택**: 구현이 쉬운 **"첫 충전 보너스 별"** 방식(결제 금액은 그대로, 별을 더 얹어줌) — 토스 결제 로직을 건드리지 않아 안전. (할인코드/금액 조정 방식은 더 복잡해 보류.)
- **광고 전용 랜딩/딥링크 + UTM**.

## 3. 측정 (없으면 광고가 장님)

- **메타 Pixel(브라우저) + Conversions API(서버) 병행** — iOS·쿠키 차단 대응, 결제/가입을 서버에서 정확히 전송.
- 이벤트: 방문(PageView) → **가입(CompleteRegistration)** → **체험완료(커스텀)** → **첫 결제(Purchase + value)**.
- **최적화 이벤트**: 초기엔 볼륨 확보를 위해 **가입**으로 최적화 → 가입이 주당 수십 건 쌓이면 **체험완료**로 상향. 결제는 value로 ROAS만 관찰.

## 4. 소재 (크리에이티브) — 3종

별콩이 캐릭터 기반 **정적 이미지·AI 생성** 중심(촬영/편집 없이 시작). 각 소재는 **피드(4:5) + 스토리(9:16)** 리사이즈. 카피 원칙: **별콩이 반말 톤 + "무료" 강조 + CTA 1개**, 페르소나 화법대로 **단정적 예언("~된다") 금지**(호기심만 자극).

1. **고민상담 훅** — "요즘 마음 복잡하지? 사주/타로로 물어봐" (고의도)
2. **무료 오늘의 운세 훅** — "무료 오늘의 운세 보고 가" (저마찰 볼륨)
3. **직접 뽑는 타로 훅** — "카드는 네가 직접 뽑아, 별콩이가 풀어줄게" (차별화·저경쟁)

> 3소재는 "모드 2(상담/운세) + 차별화 1(타로)"이 섞인 **각(角) 탐색**이다(완벽한 통제 실험 아님). 소재 3번(직접 뽑는 타로)은 사주 대비 광고가 드물고 '직접 뽑기'가 우리만의 차별점이라 저경쟁·시선 강점. 향후 **짧은 GIF/모션**을 붙이면 가장 크게 살아난다.

## 5. 타깃팅

- 지역 한국 / 연령 20~39 중심.
- **메타 Advantage+ (broad)로 시작** — 저예산에선 관심사를 좁히기보다 알고리즘이 "사주·타로 궁금한 사람"을 찾게 두는 게 유리. 데이터가 쌓이면 관심사(사주·타로·점성술·MBTI)·리타깃으로 분리.

## 6. 캠페인 구조 & 예산·학습

- **1 캠페인**(전환/최적화=가입) → **1 애드셋**(Advantage+, 일 ₩10k 전액) → **소재 3종**.
- 애드셋을 여러 개로 쪼개지 않는다(예산 분산 = 학습 실패). 소재만 넣고 메타가 승자에 예산을 몰아주게 한다.
- **최소 1~2주는 손대지 않고** 학습기 통과(성급한 수정 금지).

## 7. 성공 기준 & 의사결정 규칙 (3~4주 후)

- 관찰 지표: CTR, CPC, **가입당 비용(CPA)**, 체험완료율, 첫결제율, ROAS(관찰).
- 벤치마크는 업계 추정치 대신 **2주 뒤 우리 실측치로 갱신**(그게 이 테스트의 목적).
- 결정 규칙:
  - 가입 CPA 양호 + 체험→결제 신호 → **예산 스케일 + 최적화를 체험완료/결제로 상향 + 바이럴(B) 얹기**.
  - 클릭은 되는데 가입/체험 전환 약함 → **랜딩·오퍼 개선**.
  - CTR 낮음(클릭이 비쌈) → **소재 교체**.
  - 3~4주 LTV로 **CAC < LTV** 검증 → 되면 스케일, 안 되면 유료 축소 + 오가닉/바이럴 피벗.

## 비범위 (나중 단계)

- 바이럴(B: 공유 보너스 별로 유효 CAC↓) — 성과광고 CAC 확인 후.
- 인스타 계정 오가닉 콘텐츠 운영(C) — 병행 권장이나 별도 기획.
- 틱톡(세로 영상)·카카오(리타깃)·네이버(검색 의도)·유튜브(브랜딩) 확장.
- 소재 3번 모션(GIF) 버전.
- 관심사 타깃·리타깃 분리, 첫 결제 할인코드(금액 조정) 방식.

## 구현 범위 (다음 단계 writing-plans 에서 상세화)

1. 신규 **웰컴 별** 지급 로직(카카오 가입 시, 멱등, `star_transactions source=welcome_bonus`).
2. **첫 충전 보너스 별** 로직(결제 confirm 시 첫 충전 판정 + 보너스 지급).
3. **광고 전용 랜딩/딥링크 + UTM** (고민상담 / 오늘의운세 / 타로 draw 3경로).
4. **메타 Pixel + Conversions API** 연동 + 이벤트(가입/체험완료/첫결제) 발화.
5. (광고 계정측) 캠페인 1 · 애드셋 1(Advantage+) · 소재 3종 세팅 + 오퍼 반영 카피.
