## 0. 한 줄 요약

연애 중심 구조 전환(사이클 1~2)에 광고 소재를 정렬한다. tarot **대조군 유지** + 신규 2본(**범용연애·연애상담**)으로 **"제네릭 vs 범용연애 vs 깊은상품(연애상담)"** 3자 비교. 별콩이 **하이브리드 비주얼**(캐릭터 유지 + 연애 감정 맥락)로 브랜드 통일 + 대조군과 변수 통제.

---

## 1. 대조군 훅 겹침 점검 (§8 미해결 항목 해소)

§8이 "확인 미완"으로 남긴 리스크: *슬롯2 범용연애가 tarot 대조군과 훅이 겹치면 자기 옥션 경쟁·인크리멘탈 학습 소실.*

**확인 결과 (현행 `ad-assets/final/ad_tarot_{feed,story}.png` 실사)**:
- tarot 대조군 훅 = **"카드는 네가 직접 뽑아"** + 서브 "별콩이가 풀어줄게 · 첫 회 무료" + CTA "무료로 뽑아보기".
- 이는 **행위(직접 뽑기)·무료 훅**이지 연애/관계 앵글이 **전혀 없음** → 제네릭 타로 상품 소재.
- ∴ 슬롯2 범용연애("그 사람 마음")와 **훅 미겹침** → 리스크 해소, 슬롯2 각도 조정 불필요.

## 2. 소재 믹스 (3본 확정)

Meta 학습 볼륨 제약(광고당 주 ~50전환)상 **3본이 상한** — 구성만 교체.

| 슬롯 | 목적·측정 | 훅 카피 | 랜딩 | 포맷 |
|---|---|---|---|---|
| **1 대조군** | 벤치마크·학습 보존 | (기존 tarot **절대 무수정**) | 홈 `/` | 기존 |
| **2 범용연애** | 획득 볼륨(넓은 그물·싼 클릭·학습 볼륨) | **"그 사람은 지금 무슨 생각을 할까"** | `/start?v=love` | 피드 4:5 + 스토리 9:16 + mp4 |
| **3 연애상담** | **신상품 수요 테스트** — CVR 아닌 **패스 구매·갱신·복귀(리텐션/LTV)**, 높은 CAC 감수 | **"연애 고민 털어놓을 친구, 여기 있어"** | `/relationship` 콜드스타트 | 피드 4:5 + 스토리 9:16 + mp4 |

**공통 규격**: "첫 회 무료" 뱃지 · 다크보라+골드 톤 · 별콩이 캐릭터 유지(대조군과 "같은 캐릭터, 다른 각도").

## 3. 슬롯별 소재 상세

### 슬롯1 — 대조군 (tarot 무수정)
- 기존 `ad-assets/final/ad_tarot_{feed.png, story.png, story.mp4}` 그대로. **손대지 않음** (벤치마크·Meta 학습 보존).

### 슬롯2 — 범용연애 ("그 사람 마음")
- **앵글**: 대조군이 *행위·무료*니까 겹치지 않게 **"마음 들여다보기"(궁금증)**.
- 훅: **"그 사람은 지금 무슨 생각을 할까"** (반말·공감 톤)
- 서브: "별콩이가 카드로 비춰줄게 · 첫 회 무료"
- CTA: "무료로 마음 보기"
- **비주얼(하이브리드)**: 별콩이가 타로 카드 사이로 **은은한 하트빛을 비추는** 포즈 + 배경 골드 하트 파티클. 대조군의 "카드 부채 들기"와 구도로 구분.
- 랜딩: `/start?v=love`

### 슬롯3 — 연애상담 ("다 기억하는 친구")
- **앵글**: CVR이 아니라 **신상품 수요/리텐션**. 감정형 — **경청·지속·친구**.
- 훅: **"연애 고민 털어놓을 친구, 여기 있어"** (친구+안심 프레임)
- 서브: "매번 처음부터 말 안 해도 돼" (종목 밸류 = 지속 스레드·장기 기억을 서브로 받침)
- CTA: "내 연애 상담사 만나기"
- **비주얼(하이브리드)**: 별콩이와 **마주 앉아 차 놓고 도란도란** 들어주는 아늑한 장면(친구랑 수다 느낌). 슬롯2보다 포근한 톤.
- 랜딩: `/relationship` 콜드스타트 (§ 상위설계 5 "콜드스타트 = 마케팅 지면")

## 4. 랜딩 — `/start?v=love` (상위설계 §8 채용)

기존 `/start` variant 인프라 재사용(현재 counsel/daily/tarot + reunion/contact 존재 → `love` 추가). **연애 존 미니 홈** 구성:

1. 히어로 — 소재와 동일 카피("그 사람은 지금 무슨 생각을 할까")
2. **"네가 보고 온 그 고민" 하이라이트 카드 + 메인 CTA**
3. "다른 연애 고민이라면" — 연애 태그 6종 목록(`LOVE_TAGS`, 어느 것이든 로그인→해당 태그 고민 입력)
4. 하단 "연애 말고 다른 고민 →" 홈 링크
- 고민 입력 화면에서도 태그 변경 가능.
- ⚠️ variant 규약 준수: **`v` 우선**, `utm_content`는 어트리뷰션 전용(레거시 폴백).

## 5. 코드 팔로업 (소량 — dev, 3e prod 일괄 편입)

1. **`/start?v=love` variant 추가** — 연애 존 랜딩(위 §4). 기존 variant 스위치에 `love` 케이스.
2. **`/relationship` 콜드스타트 광고 어트리뷰션(utm/v) 수신** — 기존 `AuthBootstrap` 캡처 seam(utm/fbclid → 30일 쿠키 → 카카오 콜백이 신규 유저 `user_acquisition` insert)이 `/relationship` 콜드스타트 진입에도 적용되는지 **확인/보강**. (전역 마운트면 무코드 확인, 아니면 보강)
3. **`utm_content` 개별화** — Meta 광고 URL 파라미터(`utm_content={{ad.name}}&utm_term={{placement}}`). **코드 무관·사용자 액션**.

> 코드 팔로업 실제 구현 세부는 writing-plans에서 `/start`·`AuthBootstrap` 현행 코드 확인 후 확정.

## 6. 운영·측정 (§8 어트리뷰션 수리)

- **counsel 광고그룹 OFF** · **daily 자연종료 관찰**(릴스 CPA 높음 — 죽는 것 관찰) → 신규 3본으로 교체. (Meta 대시보드, 사용자 액션)
- URL 파라미터: `utm_content={{ad.name}}&utm_term={{placement}}` (광고 이름 = 영문 슬러그). variant는 `v` 우선.
- **신규 세트 지면 수동 지정: IG 피드 + 릴스 + 스토리** (잡지면 낭비 차단 — 기존 세트 지면은 학습 리셋 대비 실익 없어 수정 안 함).
- `ad_spend.clicks` 수기 입력 재개.

**성공지표** (배포 후 분석, 상위설계 §11 연결):
- 슬롯 단위 CAC/ROAS (utm_content 개별화 후)
- 슬롯2 범용연애: 획득 볼륨·클릭 단가·랜딩에서 태그 선택률
- **슬롯3 연애상담: 패스 구매·갱신율·복귀율(리텐션/LTV) — CVR 아님**
- 대조군 대비 인크리멘탈(훅 분리 검증)

## 7. 제작 워크플로우 + 크레딧

**기존 워크플로우 재사용** (`ad-launch-progress` 선례):
1. 힉스필드 `generate_image` (nano_banana_pro, `public/byeolkong-main.png` 레퍼런스) → 슬롯별 시안 여러 안
2. 사용자 선택 → `upscale_image` 2K
3. Figma 텍스트 조판(훅/서브/뱃지/CTA 오버레이) — 기존 파일 `T2eHOB3lchmjNMf0PdMBRu` 계열 재사용. 폰트: Figma에 Cafe24 Ssurround 없음 → 디스플레이 Jua 폴백.
4. `generate_video` (kling 시네마그래프 5초 무음 루프) → Figma 텍스트 SVG 오버레이 합성 → 스토리/릴스 mp4
5. 산출: `ad-assets/final/ad_{love,relationship}_{feed.png, story.png, story.mp4}` — 파일명 슬러그는 구현 시 확정.

**크레딧**: 잔액 **163** (starter). 영상(kling)이 크레딧을 크게 먹음 → **이미지 먼저 확보(피드+스토리 정지), 그다음 영상**. 부족 시 진행 중단하고 충전 안내(`show_plans_and_credits`).

⚠️ 스토리 영상 CTA는 하단 세이프존(하단 340px) 침범 주의 — 기존 소재가 290px로 살짝 침범한 선례. CTA ~100px 상향 배치.

## 8. 배포 타이밍

```
지금(3f): 소재 제작(dev/로컬) + 코드 팔로업(dev)
   │
3e: prod 일괄 배포 (사이클 1~3 전체 + v=love·어트리뷰션 코드)
   │
3e 이후: Meta 캠페인 소재 교체·ON (사용자 액션) — counsel OFF, 신규 3본 라이브, utm 개별화
```

- 소재 파일과 코드는 지금 만들지만 **광고 ON은 prod 이후** (랜딩이 prod에 있어야 함).
- 배포 시 페르소나 이탈지표 새 경계 커밋 기록(`persona-tuning-baseline` 패턴)은 3e의 별도 항목 — 본 문서 범위 밖.
